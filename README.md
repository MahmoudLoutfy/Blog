## Getting Started
First clone the repository

Then install the dependencies 

  ```bash
  npm install
  ```

  or if you use yarn:

  ```bash
  yarn install
  ```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Finally Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
